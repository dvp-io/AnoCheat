Bienvenue dans le centre de notifications.
Ici vous pouvez gérer de manière avancée les notifications du chat.

Si vous rencontrez un bug ou pour toute suggestion ouvrez un ticket sur le tracker du projet, merci !

Type de notification à configurer :

Configuration des notification :

Notification système
Active les notifications système, ces notifications permettent d'être notifié même quand un autre logiciel est au premier plan.

Titre personnalisé
Le titre personnalisé vous permet de rendre la notification plus discrète.

Message personnalisé
Le message personnalisé vous permet de rendre la notification plus discrète.

Image personnalisée (url ou base64)
L'image personnalisée vous permet de rendre la notification plus discrète. Si vous êtes derrière un proxy, privilégiez l'utilisation du base64 du fichier.

Masquer la notification dans le salon
Cette option permet de masquer la notification (texte italique, bleu) directement dans le salon, ne fonctionne pas pour les notifications du groupe système.

Notification sonore (url ou base64):
Permet de jouer un son lors de la notification, il est recommandé d'éviter les fichiers trop lourds. Si vous êtes derrière un proxy, privilégiez l'utilisation du base64 du fichier.
